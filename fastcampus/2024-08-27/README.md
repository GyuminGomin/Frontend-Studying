3. 특수문자 기호 이름
```
~ (틸드)
` (그레이브, 백틱)
! (익스클러메이션, 느낌표)
@ (앳 사인, 골뱅이)
^ (캐럿)
* (에스터리스크, 별)
- (하이픈, 대시, 마이너스)
_ (언더스코어, 로대시, 밑줄)
" (쿼테이션, 쌍따옴표)
' (어프로스토피, 따옴표)
. (피리어드, 닷, 마침표)
? (퀘스천)
/ (슬래시)
| (버티컬 바)
\ (백슬래시, 역슬래시)
() (퍼렌서시스, 소괄호, 괄호)
{} (브레이스, 중괄호)
[] (브래킷, 대괄호)
<> (앵글 브래킷, 꺽쇠괄호)
```
